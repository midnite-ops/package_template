# aetho-tut

A beautiful wallet connection modal for React applications.

## Installation
```bash
npm install aetho-tut
```

## Requirements

- React 16.8+
- lucide-react

## Usage
```jsx
import { Modal } from 'aetho-tut';

function App() {
  return (
    <div>
      <Modal />
    </div>
  );
}
```
